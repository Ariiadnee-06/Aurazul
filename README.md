# Aurazul – Tienda de Accesorios

## 📌 Descripción del proyecto

**Aurazul** es un sitio web de comercio electrónico desarrollado como **proyecto de la técnica Auxiliar en Desarrollo de Software**.

El proyecto consiste en una tienda virtual de accesorios donde los usuarios pueden explorar diferentes productos como **anillos, collares y pulseras**, visualizando imágenes, nombres y precios de cada artículo.

El objetivo del proyecto fue aplicar conocimientos básicos de **desarrollo web front-end**, organización de archivos y diseño de interfaces para crear una tienda digital sencilla y funcional.

---

## 🎯 Objetivos del proyecto

* Desarrollar un **e-commerce básico** para una tienda de accesorios.
* Aplicar **HTML y CSS** para el diseño de interfaces web.
* Organizar el proyecto siguiendo una **estructura clara de carpetas**.
* Diseñar un catálogo visual de productos para mejorar la experiencia del usuario.

---

## 🛠 Tecnologías utilizadas

* **HTML5** – Estructura del sitio web
* **CSS3** – Estilos y diseño visual
* **Fuentes personalizadas** para mejorar la identidad visual del proyecto

* Página principal con presentación de la tienda.
* Catálogo de productos dividido por categorías:

  * Anillos
  * Collares
  * Pulseras
* Visualización de:

  * Imágenes de productos
  * Nombre del producto
  * Precio
